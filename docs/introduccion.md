# Introducción

## Objetivo del proyecto
El objetivo del proyecto **Go Rent Bike** es ofrecer una interfaz web clara y atractiva para la visualización y alquiler de bicicletas, permitiendo al usuario consultar modelos disponibles, precios y características principales.

## Problema que resuelve

Muchos servicios de alquiler carecen de interfaces intuitivas. Este proyecto propone una solución sencilla, visual y adaptable a ordenador, tablet y móvil para mejorar la experiencia del usuario.
