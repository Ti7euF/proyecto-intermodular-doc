# Instalación

## Requisitos
- Navegador web moderno (Chrome, Firefox, Edge)

## Pasos de instalación
1. Descargar o clonar el repositorio.
2. Acceder a la carpeta del proyecto.
3. Abrir el archivo `index.html` en el navegador.
